# Glioma Grade Classification (MLOps Project)

## Overview
This project is based on my published research on glioma tumor classification using MRI images.  
It has been refactored into a production-ready ML system with API support.

## Problem Type
Binary Classification:
- Low Grade
- High Grade

## Features
- Deep learning-based image classification
- REST API using FastAPI
- Modular code structure
- Ready for Docker deployment

## Tech Stack
- Python
- TensorFlow / PyTorch
- FastAPI
- Docker (planned)

## API (Planned)
POST /predict  
Input: MRI image  
Output: Tumor grade prediction

## Data
Datasets are not included due to size and privacy.  
Recommended dataset:
- BraTS (Brain Tumor Segmentation)

## Research Background
This work is based on my previously published research in medical imaging.
